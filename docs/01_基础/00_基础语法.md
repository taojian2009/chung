### java的跨平台原理

#### JVM
java virtual machine
主要依靠JVM，在不同的平台安装一个JVM，既可以执行jav程序；

#### JRE
java runtime environment

jre是java程序的运行环境

#### JDK

java开发工具；
主要的开发工具包括: javac.exe 和java.exe


#### 三者之间的关系

jdk包含jre和开发工具；
jre包含jvm和核心类库；
jvm保证java程序的运行；
